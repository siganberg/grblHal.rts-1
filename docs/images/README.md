# README images

Images used by the flashing guide in the main [`README.md`](../../README.md).

| Filename | What it shows |
|---|---|
| `boot0-usb-dfu.webp` | Animated (~10 s): holding the BOOT0 button while plugging in USB-C to enter DFU mode. |
| `ncsender-flash.webp` | Animated (~30 s, sped up): flashing grblHAL with ncSender's Firmware tool (DFU mode → pick `.hex` → flash → done). |

These two animations cover the flashing guide. If you want to add more, drop them here and
reference them from the README.
